📘 UNIT I – 5 Marks Important Questions

1. Explain Software Engineering Layers with a neat diagram.

Meaning

Software Engineering Layers are the building blocks of software development. They help in developing quality software systematically.

Layers

  • Quality Focus – Ensures high-quality software and customer satisfaction.
  • Process – Provides step-by-step software development.
  • Methods – Includes analysis, design, coding and testing.
  • Tools – Supports development using software tools.

Example: Testing a Banking Application before release.

2. Explain Software Myths with examples.

Software Myths are false beliefs about software development that create project problems.

  • Management Myths
  • Customer Myths
  • Practitioner (Developer) Myths

Example: Adding more programmers always finishes the project faster.

3. Explain the Waterfall Model with advantages and disadvantages.

  • Requirement Analysis
  • System Design
  • Coding
  • Testing
  • Deployment
  • Maintenance

Advantages: Easy to understand, simple to manage, suitable for small projects.

Disadvantages: Difficult to change requirements, testing is done late.

Example: Library Management System.

4. Explain the Incremental Process Model with example.

Software is developed in small increments. Each increment adds new features.

  • Module-by-module development
  • Early working software
  • Easy testing
  • Flexible changes
  • Customer feedback

Example: Online Shopping Application.

5. Explain the Prototyping Model with advantages and disadvantages.

A prototype is developed first to understand user requirements before building the final software.

  • Collects user feedback
  • Improves requirements
  • Reduces errors
  • Improves quality
  • Saves development time

Example: Mobile Banking Application.
