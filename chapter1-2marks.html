📘 UNIT I – 2 Marks Important Questions

1. Define Software Engineering.

Answer:
Software Engineering is the systematic process of designing, developing, testing, deploying, and maintaining software.

Example: Developing a College Management System.

2. What is Software?

Answer:
Software is a collection of programs, data, and instructions that tell a computer how to perform tasks.

Example: MS Word, Google Chrome, Windows.

3. What are Software Myths?

Answer:
Software myths are false beliefs or misconceptions about software development that can cause project problems.

Example: Believing that adding more programmers always finishes the project faster.

4. List the Software Engineering Layers.

Answer:

1. Quality Focus
2. Process
3. Methods
4. Tools

Example: Using testing tools to improve software quality.

5. What is a Process Model?

Answer:
A Process Model is a step-by-step method used to develop software in an organized way.

Example: Waterfall Model.

6. Define Waterfall Model.

Answer:
The Waterfall Model is a sequential software development model where each phase is completed before the next phase begins.

Example: Library Management System.

7. Define Incremental Process Model.

Answer:
The Incremental Process Model develops software in small parts (increments), with each increment adding new features.

Example: Online Shopping App.

8. What is the Prototyping Model?

Answer:
The Prototyping Model creates a sample (prototype) of the software to collect user feedback before final development.

Example: Food Delivery App Interface.

9. What is the Spiral Model?

Answer:
The Spiral Model is a risk-based software development model that develops software in repeated cycles (spirals).

Example: Banking Software.

10. List any four Software Application Domains.

Answer:

1. Business Software
2. Medical & Healthcare Software
3. Finance Software
4. Social Media Software

Examples:
Business – Billing System
Medical – Hospital Management System
Finance – ATM Software
Social Media – WhatsApp
